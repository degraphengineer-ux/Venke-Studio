## 📸 About The Project

**Venkes Studio** is a motion-driven photography portfolio that redefines the traditional gallery format. Rather than presenting static images, it creates a fully immersive visual experience that fuses cinematic storytelling, sophisticated interaction design, and purposeful motion.

The project positions photography as an _experience_ — one that evokes emotion, guides attention, and invites deeper engagement.

---

## 🎯 Key Features

- Cinematic hero section with powerful visual identity
- Interactive folder-style gallery navigation
- Custom cursor interactions and click sound feedback
- GSAP-powered animations with ScrollTrigger
- Smooth page transitions and scroll-triggered effects
- Fully responsive, minimal dark-themed design

---

## 🧠 Design Philosophy

Built on four foundational principles:

- **Less, but better** — Every element serves a purpose; unnecessary noise is removed
- **Emotion-first design** — Feeling takes precedence over conventional structure
- **Motion with purpose** — Animations direct attention and enhance storytelling
- **Experience over layout** — Interaction itself becomes the product

---

## 🛠️ Tech Stack

- **HTML5** – Semantic, accessible structure
- **CSS3** – Custom design system and responsive architecture
- **JavaScript (ES6+)** – Interaction and logic layer
- **GSAP + ScrollTrigger** – High-performance animation engine
